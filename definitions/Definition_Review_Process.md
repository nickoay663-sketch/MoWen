# Definition Review Process

Every definition must pass the following review before becoming an official MoWen definition.

1. Candidate
2. Definition
3. Scope
4. Reasoning
5. Counterexample
6. Responsibility
7. Approved

---

任何定义，在成为莫问正式定义之前，都必须经过完整审核。
