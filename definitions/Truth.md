# Truth

## Definition

Truth is a statement that corresponds to reality.

A statement is not true because it is believed.

A statement is not true because it is repeated.

A statement is true only when it corresponds to reality.

---

## Runtime

Truth is not generated.

Truth is verified through Runtime execution.

---

## Relationship

Truth requires:

- Testimony
- Definition
- Evidence
- Correspondence

Without correspondence,

truth cannot be established.
