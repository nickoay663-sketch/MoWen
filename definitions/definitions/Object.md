# Object

## Definition

An object is anything that a testimony refers to.

An object may be physical, conceptual, individual, collective or abstract.

---

对象，是证词所指向的任何事物。

对象可以是实体、概念、个人、群体或抽象事物。

---

## Runtime Meaning

Recognition identifies objects before retrieving definitions.

Without identified objects, Runtime cannot continue reliably.

---

Recognition 在获取定义之前先识别对象。

没有对象，Runtime 无法可靠继续。

---

## Requirements

Every object must be identifiable.

Every object must be distinguishable from other objects.

---

每一个对象都必须能够被识别。

每一个对象都必须能够与其他对象区分。

---

## Responsibility

MoWen identifies objects.

It does not invent them.

---

莫问识别对象。

不会创造对象。
