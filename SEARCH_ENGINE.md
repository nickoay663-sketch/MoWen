# Search Engine

## Purpose

The Search Engine retrieves existing knowledge before Runtime analysis.

MoWen never assumes a definition.

MoWen searches first.

---

## Search Sources

The Search Engine may retrieve information from:

- Local Knowledge Base
- Definition Library
- Evidence Library
- Laws
- Scientific Papers
- Historical Records
- Dictionaries
- Encyclopedias
- Trusted Websites

---

## Runtime Position

Recognition

↓

Definition

↓

Search

↓

Evidence

↓

Correspondence

↓

Reasoning

↓

Responsibility

↓

Generator

---

## Search Report

Runtime always displays:

- Search completed
- Sources searched
- Results found
- Results not found

Search never hides missing information.

"No result" is also a Runtime result.

---

## Responsibility

Search does not determine truth.

Search only reports available knowledge.

Truth is established during Correspondence.
