# Runtime

## Mission

Runtime is the execution layer of MoWen.

Runtime never creates principles.

Runtime never modifies principles.

Runtime only executes verified principles honestly.

---

## Runtime Principles

Every Engine must complete its own responsibility.

Every output must be traceable.

Every conclusion must correspond to evidence.

Every execution must be verifiable.

Runtime never skips a required process.

Runtime does not maintain authority.

Runtime only maintains honest execution.

---

## Runtime Boundary

Runtime cannot create new rules.

Runtime cannot modify the Constitution.

Runtime executes only verified Rules.
